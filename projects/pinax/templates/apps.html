External

These already exist as independent apps. Some pre-existed Pinax, some were written especially for it.

django-notification
user notification management
django-email-confirmation
simple email verification
django-command-extensions
useful management commands
django-robots
for managing robots.txt files
django-databasetemplateloader
loader for templates from the database
django-friends
friendship, contact and invitation management
django-mailer
mail queuing and management
django-messages
private messages between users
django-announcements
site announcements
django-oembed
support for oembeded links
django-logging
debugging and logging
django-db-log
exception logging to the database
django-pagination
easy pagination
django-threadedcomments
threaded comments
django-wikiapp
simple wiki
django-evolution
db schema evolution
django-timezones
timezone localization
