Changes in version 4.1

Here is a summary of the major changes in Universal Feed Parser 4.1.

  • Support for the Universal Encoding Detector to autodetect character encoding of feeds that declare their encoding incorrectly or don't declare it at all. See Character Encoding Detection for details of when this gets called.
  • Universal Feed Parser no longer sets a default socket timeout (bug 1392140). If you were relying on this feature, you will need to call socket.setdefaulttimeout(TIMEOUT_IN_SECONDS) yourself.
