Package openid.consumer

This package contains the portions of the library used only when implementing an OpenID consumer.
Submodules
  • consumer: OpenID support for Relying Parties (aka Consumers).
  • discover: Functions to discover OpenID endpoints from identifiers.
  • html_parse: This module implements a VERY limited parser that finds <link> tags in the head of HTML or XHTML documents and parses out their attributes according to the OpenID spec.
